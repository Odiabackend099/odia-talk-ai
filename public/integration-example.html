ODIA Voice Chat Widget Integration

Quick Integration Steps:

  1. Add the container div to your HTML page
  2. Include the widget script
  3. Configure your ElevenLabs API key (optional)

Step 1: Add Container

<div id="odia-chat-container"></div>

Step 2: Include Widget Script

<script src="https://your-domain.com/dist/widget/widget.js" type="module"></script>

Complete Example:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Your Website</title> </head> <body> <!-- Your website content --> <h1>Welcome to Your Website</h1> <p>Your content here...</p> <!-- ODIA Widget Container --> <div id="odia-chat-container"></div> <!-- ODIA Widget Script --> <script src="https://your-domain.com/dist/widget/widget.js" type="module"></script> </body> </html>

Configuration (Optional):

<script> // Optional: Configure before loading window.ODIAConfig = { apiKey: 'your-elevenlabs-api-key', agentId: 'your-agent-id' }; </script>

Note: The widget will appear as a floating chat button in the bottom-right corner of your website. Users can click it to start a voice conversation with your AI agent.
